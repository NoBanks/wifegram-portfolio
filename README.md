# Wifegram

**Never miss a chance to make her smile.**

An AI co-writer concierge that helps busy husbands send a thoughtful, personalized message to their wife every day. You stay the hero. Wifegram handles the execution.

Live at [wifegram.livingagentic.me](https://wifegram.livingagentic.me).

---

## How it works

1. **Build her Master File in 4 minutes.** Her name, pet names, what she's into, what she's carrying right now, inside jokes, love language, what to never mention.
2. **Pick the vibe and the times.** Encouragement, humor, romantic, check-in. Morning, afternoon, night.
3. **You approve every send.** 15 minutes before each scheduled fire, the bot DMs you the draft plus 4 image options. Tap Send, or rewrite the text with one-tap iteration buttons (More casual, Shorter, Add inside joke, More specific, Rewrite).
4. **It lands in her chat as if from you.** Telegram delivery. Text plus your chosen image. No watermark, no footer.

## Privacy

- App-level encryption on sensitive Master File fields before storage. The key never sits next to the file.
- Do-not-mention list stored as one-way hashes, used as an inference filter so flagged topics never reach generation.
- Per-account row-level isolation. Data never trained on, never sold, never shared across accounts.
